# Laravel-Crud

simple CRUD project created with laravel v7

## Requirements

| no | name | version |
| ------------- | ------------- | ------------- |
| 1 | apache / nginx | * |
| 2 | php | >= 7.3 |
| 3 | laravel | 7 |
| 4 | mariaDB | >= 10 |

## Installation

* clone this project
```
git clone https://github.com/adiyansahcode/laravel-crud.git
```

* run composer
```
composer install
```

* run Migration and Seeder
```
php artisan migrate:refresh --seed
```

* done, just try run your project in browser
